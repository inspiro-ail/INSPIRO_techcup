# INSPIRO_techcup
The list of projects which were made by our team members.

Наша команда **INSPIRO** активно занимается созданием различных проектов с применением электроники, механики, технологий Machine Vision, ML/AI.
Данные проекты были признаны лучшими на республиканских и международных аренах.
Список проектов и их описание:
1. **Lorax animatronic robot** 2026
   1. Робот-аниматронник, созданный для творческого робототехнического шоу для участия на международных соревнованиях по робототехнике в Сингапуре RoboCup Asia-Pacific.
   2. Включает в себя:
      1. Сихнронизация порядка событий
      2. Автономное перемещение по сцене через Computer Vision
      3. Реалистичная мимика, сложная кинеатика движений
      4. Мобильная платформа с всенаправленными Omni колесами
      5. Локальная точка доступа на базе Raspberry Pi 5
      6. Обученная голосовая модель через ИИ
      7. Анимация через собственный программный движок

2. **AI Robot Tree** 2024
   1. Робот-аниматроник в виде дерева, способный видеть и говорить с помощью ИИ, вошедший в топ лучших проектов в Нидерландах на RoboCup Eindhoven 2024 и 2-ое место в качестве исследовательского проекта.
   2. Использованные библиотеки:
      1. VOSK - для локального Speech Recognition
      2. OpenCV - для Object Detection
      3. Edge Impulse - для быстрого Face Recognition
      4. Pyttsx3 - для озвучивания речи
      5. Llama2 7B - локальная LLM для генерации речи
      6. Raspberry Pi OS

6. **Modular AI-Robotics Station "MiniFab"** 2026
   1. Аппаратно-программный комплекс модульной роботизированной станции для сбора датасетов и обучения сложных манипуляционных задач, занявший 1-ое место на Infomatrix Asia 2026
   2. Включает в себя:
      1. манипулятор: собственный кастомный робот-манипулятор с 6 степенями свободы (6-DoF)
      2. телеуправление: система ручного контроля и демонстрации движений с помощью 3D-мыши
      3. компьютерное зрение: мультикамерная система для фиксации окружения и действий робота
      4. ai-вычисления: выделенный ПК для инференса VLA-моделей (Vision-Language-Action) и агрегации данных
      5. робототехнический фреймворк: модифицированная и кастомизированная под задачи проекта библиотека LeRobot от Hugging Face
      6. сбор данных: пайплайн для записи демонстраций (dataset collection) в реальном времени для последующего обучения
      7. hardware: модульный корпус станции, сервоприводы манипулятора, контроллеры связи и сенсоры
      8. backend / ml engine: Python, PyTorch (в экосистеме LeRobot), библиотеки для работы с пространственным вводом 3D-мыши и видеопотоками с камер

        
7. **Autonomous Interactive Robot "Toothless"** 2025
   1. Робот-аниматроник в виде дракона, занявший первое место на международном соревновании RoboCup Asia-Pasific Abu Dhabi
   2. Включает в себя:
      1. аниматроника: система независимых моторов и актуаторов для мимики (эмоций), движения крыльев и жестов тела
      2. dialogue ai: модуль искусственного интеллекта для обработки естественного языка и речевого взаимодействия с актером/публикой
      3. analytical ai: модуль диагностического мышления для симуляции сценария «медицинского осмотра» дракона
      4. computer vision (object recognition ai): система распознавания образов для трекинга границ физических объектов и ориентации в пространстве
      5. дополненная реальность (ar): ar-дисплей для визуализации «рентгена» внутреннего состояния робота в реальном времени
      6. автономное поведение: система принятия решений на основе деревьев поведений (behavior trees) и условной логики
      7. hardware: бортовой управляющий компьютер Raspberry Pi, модульные контроллеры приводов, микрофоны и камеры
      8. логика и навигация: алгоритмы автономного обхода препятствий, невербальной обратной связи и синхронизации с таймлайном шоу
